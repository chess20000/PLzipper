**PLzipper**<br>
一个小工具，用来压缩你的个人图库，让你在存储空间不够的手机上也能访问一个微缩版本的照片库。<br>
**使用方法：**<br>
先安装python<br>
运行plzipper.py<br>
输入你的个人图库的文件夹，比如D:/Pictures<br>
输入你需要输出的文件夹，没有什么要求，只要是程序能够访问的一个空文件夹即可，如果选择的不是空文件夹，程序会提示是否需要帮忙清空，请谨慎操作。**不明白的话一定不要随便输入yes**丢失文件后果自负。<br>
生成的微缩版本内目录结构和原目录结构相同，区别只是图片被压缩了，**名称相同**的raw+jpg会被合并。或者，名称相同的只是格式不同的照片生成出来的微缩版本也会变成一张，请注意。至于是哪一张，作者也不太清楚。<br>
**其他用法：**<br>
plzipper <输入的待处理文件夹> <输出的目标空文件夹>
